Snippet Anatomy

The snippet has 2 tags:

  1. Paragraph Tag <p> HTML Paragraph Reference
  2. Anchor Tag <a> HTML Anchor Reference

There are a few other things to note about the snippet. In that code snippet the anchor tag (<a>) is a child of the the paragraph tag (<p>).

Tag Anatomy

Tags are elements and have most contain opening and closing tags. The difference between <p> and </p> is that the former is an opening tag and the latter is a closing tag. Also tags can have attributes which are extra bits of information. The anchor tag <a> in the snippet has the following attributes:

  1. Href Attribute: The purpose of this attrubute is to specify the link's destination. When clicked it will take you to that destination. The value for the attribute in the snippet is "https://google.com". When clicked it will take you to that site.
  2. ID Attribute: The purpose of this attribute is specify a unique ID for an HMTL element. The ID attribute is most used to point to a style in a CSS stylesheet. The value for the attribute in this snippet is "the-best-link". There is no stylesheet so nothing will happen.
  3. Class Attribute: The purpose of this attribute is to specify one or more class names for an element. It is used similar to the ID attribute in that it is most used to point to a class in the CSS stylesheet. The value for the attribute in this snippet is "banana". There is no stylesheet so nothing will happen.
  4. Target Attribute: The purpose of this attribute is to specify a name or a keyword that indicates where to display the response that is received after submitting the form. In this case since the value of the attribute is "_blank", the page will open in a new tab or window.
