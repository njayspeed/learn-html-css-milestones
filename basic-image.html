Basic Image Editing and Viewing

Welcome to the Basic Image Editing and Viewing page. This page will serve to help you add basic images to a webpage. I will add an image in the assets/img directory and display it on the webpage using the <img> tag.

The code for this page lives in basic-image.html. There are two ways to see its contents:

  1. Open basic-image.html in VS Code or another editor
  2. Right-click anywhere on this page and select View Source
